Connect Your AI IDE to Mimir via MCP

Add the config below to your IDE, restart it, and your AI assistant will have direct access to all playbooks.

File: ~/.codeium/windsurf/mcp_config.json

{
  "mcpServers": {
    "mimir": {
      "command": "/absolute/path/to/mimir/venv/bin/python",
      "args": [
        "/absolute/path/to/mimir/manage.py",
        "mcp_server",
        "--user=admin"
      ],
      "env": {
        "DJANGO_SETTINGS_MODULE": "mimir.settings",
        "PYTHONPATH": "/absolute/path/to/mimir",
        "MIMIR_MCP_MODE": "1"
      },
      "disabled": false
    }
  }
}
