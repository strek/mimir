Connect Your AI IDE to Mimir via MCP

Two steps: register to get your token, then add the config snippet to your IDE.

Step 1 — Register and get your token

Register at mimir.featurefactory.io — your authentication token is shown on the confirmation page. Copy and save it.

Step 2 — Add to your IDE config

The config below pulls the latest Mimir MCP container from ECR and runs it with your token. No local install needed — Docker handles everything.

File: ~/.codeium/windsurf/mcp_config.json

{
  "mcpServers": {
    "mimir": {
      "command": "docker",
      "args": [
        "run", "--rm", "-i",
        "-e", "BASE_URL=https://mimir.featurefactory.io",
        "-e", "TOKEN=<your-token>",
        "-e", "MCP_TRANSPORT=stdio",
        "public.ecr.aws/h1b6q4p0/mimir-mcp-facade:latest"
      ]
    }
  }
}
